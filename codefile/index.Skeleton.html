"Cops and robbers" world

A cop chases a robber across a 1-dimensional grid. If the robber is caught, it resets to start position. Score = Number of times robber gets caught in a run.

SkeletonWorld.java

Source code of sample Mind

SkeletonMind.java

State

State = "c,r" where:
  • c = position of cop on the 1-dimensional grid, takes values 0 to ( SkeletonWorld.GRID_SIZE - 1 )
  • r = position of robber

Action

Action = "i" where i takes values:
  • SkeletonWorld.ACTION_LEFT (move left)
  • SkeletonWorld.ACTION_RIGHT (move right)

Minds may return extra information (which can be read by other Minds, but which the World will ignore):
action = "i,w1,w2,...,wn"

Score

Score = "s1,s2" where:
  • s1 = number of times robber was caught (primary score)
  • s2 = number of times robber was caught due to our action, not robber's action (secondary score)
